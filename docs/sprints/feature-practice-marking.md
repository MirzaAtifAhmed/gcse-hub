# Feature: Maths Practice + Marking Foundation

## Included

- Practice answer submission.
- Simple exact-match marking.
- Attempt storage.
- Worked solution shown after answer.
- Mark scheme shown after answer.

## Current marking behaviour

The first version uses normalised exact matching:

- ignores spaces
- ignores case
- ignores commas
- ignores £ symbols

Later versions will support equivalent fractions, algebraic equivalence and multi-step partial marking.
