# Feature: Exam Generator Foundation

## Included

- 30 minute generated maths paper.
- 45 minute generated maths paper.
- 60 minute generated maths paper.
- Topic breakdown.
- Total marks.
- Worked solutions available after answering.

## Endpoint

```text
GET /api/exams/generate?year=8&durationMinutes=60
```

## Notes

This is the first exam generator foundation. Later versions will persist exam attempts, add timers, save answers and generate parent reports.
