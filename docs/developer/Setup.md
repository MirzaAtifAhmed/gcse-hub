# Local Setup

## Yarn Classic

This project uses Yarn 1.22.22.

```bash
npm install -g yarn@1.22.22
yarn --version
```

Expected:

```text
1.22.22
```

## Install

```bash
yarn install
```

## Environment

```bash
cp apps/api/.env.example apps/api/.env
cp apps/web/.env.example apps/web/.env
```

## Run

```bash
yarn dev
```
