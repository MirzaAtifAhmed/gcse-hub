# Feature Bundle: Curriculum + Maths Question Foundation

Apply this bundle after `feat: add parent and student management`.

## Apply
Extract this ZIP over your current project using Merge/Replace.

## Run
```bash
yarn typecheck
yarn build
yarn workspace @gcse-hub/api seed
```

## Commit
```bash
git add .
git commit -m "feat: add curriculum and maths question foundation"
```

## Included
- Curriculum topic model
- Skill model
- Question template model
- Seed data for Maths Year 7/8 topics and skills
- First GCSE-style maths question templates
- Worked solutions and mark schemes stored with templates
- Curriculum API endpoints
- Maths practice API endpoint returning question templates
- Student dashboard practice preview with solutions
